# Image Data Augmentation and Image Classification

This project was completed as part of the Summer Internship Program 2026 conducted by IDEAS (Institute of Data Engineering, Analytics and Science Foundation).

## Project Objectives

* Perform image preprocessing using OpenCV.
* Apply image augmentation techniques such as grayscale conversion, image shifting, resizing, and rotation.
* Create image transformation pipelines using Torchvision.
* Load and process Cat-Dog image datasets using ImageFolder and DataLoader.
* Implement K-Means clustering on the MNIST handwritten digits dataset.
* Evaluate clustering performance using the Macro F1 Score.

## Technologies Used

* Python
* NumPy
* OpenCV
* PyTorch
* Torchvision
* Scikit-learn

## Tasks Covered

1. Image Loading and Visualization
2. Grayscale Image Conversion
3. Image Saving
4. Image Translation (Shifting)
5. Image Resizing
6. Image Rotation
7. Data Augmentation Pipeline Creation
8. ImageFolder Dataset Loading
9. DataLoader Implementation
10. MNIST Dataset Processing
11. K-Means Clustering
12. F1 Score Evaluation

## Outcome

The project demonstrates fundamental computer vision and machine learning techniques, including image augmentation, dataset handling, clustering, and performance evaluation.
